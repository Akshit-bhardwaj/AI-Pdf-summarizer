# 📄 AI PDF Summarizer

An AI-powered PDF Summarizer built using **Python**, **LangChain**, **Mistral AI**, and **Streamlit**. Upload any PDF document and receive an intelligent summary generated by a Large Language Model (LLM).

---

## Features

- 📂 Upload PDF documents
- 🤖 AI-powered summarization using Mistral AI
- 📖 Automatic PDF text extraction
- ⚡ Fast document processing
- 🎨 Interactive Streamlit UI
- 🔒 Environment variable support using `.env`
- 🧩 Modular and scalable codebase

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/AI-PDF-Summarizer.git
cd AI-PDF-Summarizer
```

### Create Virtual Environment

Windows

```bash
python -m venv .venv
```

Activate

```bash
.venv\Scripts\activate
```

macOS/Linux

```bash
source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Environment Variables

Create a `.env` file in the project root.

```env
MISTRAL_API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

---

## 📌 How It Works

1. Upload a PDF document.
2. Extract text from the PDF using PyPDFLoader.
3. Send the extracted content to Mistral AI.
4. Generate an AI-powered summary.
5. Display the summary in the Streamlit interface.
